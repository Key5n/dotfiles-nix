---
description: Commit and Open a draft Pull Request
---

Open a draft Pull Request of the current branch.
Use the outline below for the Pull Request, but adapt it as needed.

Requirements:

- If the repository has a Pull Request template, use it as the base.
- If there are no commits on the current branch, commit the worktree with a concise title.
- Use the commit message as the Pull Request title.
- Make the Pull Request detailed enough to describe the changes clearly.
- Maintain a professional tone.
- Write the Pull Request in Japanese.

Recommended Sections

- "背景"
- "変更概要"
  - Explain this section as clearly and precisely as possible.
