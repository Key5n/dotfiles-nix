# NixOS Desktop Setup

Installation

```sh
sudo nixos-rebuild switch --flake github:Key5n/dotfiles-nix#nixos-desktop
```

## Dual Boot with Windows

- You need to copy the Windows boot loader to the Linux ESP. Then `systemd-boot` auto detection works.
